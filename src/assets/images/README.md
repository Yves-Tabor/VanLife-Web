# Professional UI Icons

This directory contains professional icons for the VanLife application.

## Required Icons:

### Navigation Icons:
- `home-icon.png` - Home/Logo icon
- `van-icon.png` - Van/Vehicle icon  
- `host-icon.png` - Host/Management icon
- `about-icon.png` - About/Info icon
- `login-icon.png` - Login/User icon
- `avatar-icon.png` - User profile avatar (already exists)

## Icon Specifications:
- Size: 24x24px for navigation, 32x32px for larger displays
- Format: PNG with transparency
- Style: Professional, modern, consistent design
- Colors: Monochrome or brand-appropriate colors

## Usage:
Icons are imported and used in the NavBar component for professional navigation appearance.

## Sources:
You can get professional icons from:
- Font Awesome (PNG versions)
- Material Design Icons
- Feather Icons
- Heroicons
- Custom designed icons
